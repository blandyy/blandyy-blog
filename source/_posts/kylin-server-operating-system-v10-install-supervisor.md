## 背景

产品在做国产化适配，我负责对安装包，docker依赖等做适配，记录下安装supervisor的步骤

## 软件包

安装supervisor需要先安装setuptools，这些都是python库的形式进行安装，麒麟V10内置了python3.7，可以省去安装python的步骤。

setuptools地址：[https://pypi.org/project/setuptools/](https://pypi.org/project/setuptools/)

supervisor地址：[https://pypi.org/project/supervisor/](https://pypi.org/project/supervisor/)

下载他们的tgz包即可，本文以setuptools-67.7.2和supervisor-4.2.5版本作为示范。

<!-- more -->

## 安装步骤

### 安装setuptools

```bash
tar -zxf setuptools-67.7.2.tar.gz
cd setuptools-67.7.2/ && python3 setup.py install
```

### 安装supervisor

```bash
tar -zxf supervisor-4.2.5.tar.gz
cd setuptools-67.7.2/ && python3 setup.py install
```

```bash
vi /etc/supervisord.conf
```

### 创建对应目录

```bash
mkdir -p /etc/supervisord.d/
mkdir -p /var/log/supervisor/
mkdir -p /run/supervisor/
```

### 启动服务

```bash
supervisord -c /etc/supervisord.conf
```
